# chzzkGamble
(2024.08.28 초안)

## 개요
해당 프로젝트는 웹 방송 어플리케이션 '치지직' API 활용하여 같이 즐길 수 있는 겜블(혹은 게임)을 만드는 것을 목적으로 하고 있습니다.


## 갬블 목록
- 도네이션 돌림판
  - 스트리머 채팅창에 도네이션 내용도 같이 올라오는 점을 활용
  - 특정 시간 동안 받은 도네이션을 돌림판 요소로 전환한다.
  1. 시간은 지정 가능하며, 최대 30분 제한
  2. 1000 치즈 당 1표
  3. 도네이션 내용에 키워드를 포함하는지 여부에 따라 돌림판 요소로 전환
  - 고유 `ChannelId`가 필요함
